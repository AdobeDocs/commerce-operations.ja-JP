# [!DNL Recommendations]

[!DNL Site-Wide Analysis Tool] ページの[!UICONTROL Recommendations page]には、次の機能が用意されています。

* Adobe Commerce サイトの全体的な健全性を監視および改善するのに役立つ[SWAT ヘルスインデックススコア ](swat-health-index.md)
* サイトで検出された問題に対処するためのベストプラクティスにもとづいて、レコメンデーションリストを作成します。

推奨事項は、POからP4までの優先度で並べ替えられ、POが重要でP4が低くなります。 調査結果には、説明、推奨、サイトへの影響、根本原因、シナリオ/前提条件、期待される結果、使用されたツールなどが含まれます。

## 使用するタイミング

[!UICONTROL Recommendations] ページを使用して、サイトの健全性を向上させるための推奨事項を表示および実装します。

## Adobe Workfrontの利点

* サイトパフォーマンスを向上させるためのベストプラクティスを解説します。
* 優先度に基づいて、リストされた推奨事項を追跡および実装します。
* SWAT Health Index Scoreをモニタリングすることで、サイトの改善の進捗状況を追跡できます。

## ベストプラクティス

[!DNL Site-Wide Analysis Tool] [!UICONTROL Recommendations] ページには、優先度がP0からP4の推奨事項が一覧表示されます。

* P0は重要です
* P1緊急
* P2が高い
* P3は中程度
* P4が低い。

このページを監視してこれらの推奨事項を確認し、優先順位に基づいて実装することで、最適なサイトパフォーマンスを確保できます。
