# ACP2E-4626:Storefront JavaScript ファイルの読み込みが2回発生すると、読み込みが重複し、動作が不安定になる

ACP2E-4626 パッチは、ストアフロント JavaScript ファイルがリクエストされ、2回実行される問題を修正し、断続的な重複ロードと不安定な動作を引き起こします。修正後、JavaScriptの依存関係は1回だけ読み込まれ、既存のMixin機能を維持しながら重複するリクエストを排除します。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.80がインストールされている場合に利用できます。 パッチ IDはACP2E-4626です。 この問題は、Adobe Commerce 2.4.9で修正される予定です。

## 影響を受ける製品とバージョン

**パッチはAdobe Commerceのバージョン**&#x200B;用に作成されました

* Adobe Commerce（すべてのデプロイメント方法） 2.4.8-p3

**Adobe Commerceのバージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7 - 2.4.8-p5

>[!NOTE]
>
>このパッチは、新しい[!DNL Quality Patches Tool] リリースを含む他のバージョンに適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]：パッチの検索ページ &#x200B;](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)で互換性を確認します。 パッチ IDを検索キーワードとして使用して、パッチを検索します。

## イシュー

一部のストアフロント JavaScript ファイルがリクエストされ、2回実行され、断続的な重複ロードと不安定な動作が発生する問題。 修正後、依存関係は1回だけ読み込まれ、既存のMixin機能を維持しながら重複するリクエストを排除します。」

<u>前提条件</u>:

1. MSIがインストールされています。

<u>複製する手順</u>:

1. ブラウザーでストアフロントページを開きます。
1. ブラウザー&#x200B;**[!UICONTROL DevTools]**&#x200B;を開き、**[!UICONTROL Console]** タブを選択します。
1. 次のコマンドを実行します。

   ```
   require.config({ deps: ['mage/common'] });
   ```

<u>期待される結果</u>:

重複した負荷は発生しません。 呼び出しは必要な関数を返し、新しいスクリプトリクエストは生成されません。

<u>実際の結果</u>:

新しいリクエストが&#x200B;**[!UICONTROL Network]** タブに表示されます（例：mage/common.js、requirejs/domReady.js、jquery.js）。
