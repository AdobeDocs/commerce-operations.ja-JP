# ACSD-69308: `special_price`がweb サイト レベルで設定されている場合、カタログ価格ルールが適用されない

ACSD-69308 パッチは、`special_price`がweb サイトレベルでのみ設定された場合（「すべてのストアビュー」ではなく）にカタログ価格ルールが適用されなかった問題を修正します。 修正後、最初にweb サイトのデフォルトのストアを確認することで、カタログ価格ルールが正しく適用されます。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.76がインストールされている場合に利用できます。 パッチ IDはACSD-69308です。 この問題は、Adobe Commerce 2.4.9で修正される予定です。

## 影響を受ける製品とバージョン

**パッチはAdobe Commerceのバージョン**&#x200B;用に作成されました

* Adobe Commerce（すべてのデプロイメント方法） 2.4.8

**Adobe Commerceのバージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>このパッチは、新しい[!DNL Quality Patches Tool] リリースを含む他のバージョンに適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]：パッチの検索ページ &#x200B;](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)で互換性を確認します。 パッチ IDを検索キーワードとして使用して、パッチを検索します。

## イシュー

`special_price`がweb サイト レベルでのみ設定されている場合、カタログ価格ルールは適用されませんでした。 特定のストアビューに割り当てられた商品の価格がシステムによって正しく認識されなかったため、価格に一貫性がなくなりました。

<u>複製する手順</u>:

1. Admin パネルにログインします。
1. カスタム Web サイト、ストアビューおよびストアビュー（web2）を作成します。
1. **[!UICONTROL Catalog Price Scope]**&#x200B;を&#x200B;*Web サイト*&#x200B;に設定します。
1. `special_price`属性に&#x200B;**[!UICONTROL Use for Promo Rule Conditions]**&#x200B;を&#x200B;*はい*&#x200B;に設定します。
1. 次のカタログ価格ルールを作成します。

```yaml
Website: web2
Conditions:
Special Price  greater than  15
Special Price  less than  50
Actions:
Apply: Apply as percentage of original
Discount Amount: 100
```

1. **web2** ストアビューにのみ割り当てられた2つの製品を作成します。

製品01:

```text
Set Advanced Pricing:
All Store View Scope: 30.00
web2 scope: 40.00
```

製品02:

```text
Set Advanced Pricing:
All Store View Scope: keep empty
web2 scope: 40.00
```

1. **web2** ストアビューの下にあるストアフロントで各製品を再インデックスして読み込みます。

<u>期待される結果</u>:

カタログ価格ルールは、web2 ストアビューに適用する必要があります。 特別価格がカタログ ルール条件と一致するため、製品02には&#x200B;*0.00*&#x200B;の価格が表示されます。

<u>実際の結果</u>:

* 製品01：価格は&#x200B;*0.00*&#x200B;と表示され、カタログルールの条件と一致します。
* 製品02：価格は&#x200B;*40.00*&#x200B;と表示されます。これは&#x200B;**が予想されていません**。カタログルールは、web サイトのレベルで設定された特別価格に基づいて適用する必要があります。
