# ACSD-69308:`special_price` が web サイトレベルで設定されている場合、カタログ価格ルールが適用されない

ACSD-69308 パッチは、Web サイトレベル（「すべてのストアビュー」では `special_price` く）で設定した場合に、カタログの価格ルールが適用されない問題を修正しました。 修正後、最初に web サイトのデフォルトストアを確認することで、カタログの価格ルールが正しく適用されます。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.76 がインストールされている場合に使用できます。 パッチ ID は ACSD-69308 です。 この問題はAdobe Commerce 2.4.9 で修正される予定であることに注意してください。

## 影響を受ける製品とバージョン

**Adobe Commerce バージョン用のパッチが作成されます。**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.8

**Adobe Commerce バージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>このパッチは、新しい [!DNL Quality Patches Tool] リリースを含む他のバージョンにも適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html) で互換性を確認します。 パッチ ID を検索キーワードとして使用して、パッチを見つけます。

## 問題

カタログ価格ルールは、web サイトレベル `special_price` のみ設定されている場合は適用されませんでした。 システムが特定の店舗表示に割り当てられた製品の価格を正しく認識しなかったため、価格に一貫性がなかった。

<u> 再現手順 </u>:

1. 管理パネルにログインします。
1. カスタム web サイト、ストア、ストア表示（web2）を作成します。
1. **[!UICONTROL Catalog Price Scope]** を *Web サイト* に設定します。
1. **[!UICONTROL Use for Promo Rule Conditions]** 属性の *を* はい `special_price` に設定します。
1. 次のカタログ価格ルールを作成します。

```
Website: web2
Conditions:
Special Price  greater than  15
Special Price  less than  50
Actions:
Apply: Apply as percentage of original
Discount Amount: 100
```

1. **web2** ストア表示のみに割り当てられた 2 つの製品を作成します。

製品 01:

```
Set Advanced Pricing:
All Store View Scope: 30.00
web2 scope: 40.00
```

製品 02:

```
Set Advanced Pricing:
All Store View Scope: keep empty
web2 scope: 40.00
```

1. **web2** ストア表示のストアフロントで各製品のインデックスを再作成して読み込みます。

<u> 期待される結果 </u>:

カタログ価格ルールを web2 ストア表示に適用する必要があります。 商品 02 の特別価格はカタログルールの条件に一致するため、その商品 02 には *0.00* の価格が表示されます。

<u> 実際の結果 </u>:

* 製品 01：価格は、カタログ・ルール条件に一致する *0.00* と表示されます。
* 製品 02：カタログルールは、web サイトレベルで設定された特別な価格に基づいて適用する必要があるので、価格は *期待されていない&#x200B;***40.00** と表示されます。
