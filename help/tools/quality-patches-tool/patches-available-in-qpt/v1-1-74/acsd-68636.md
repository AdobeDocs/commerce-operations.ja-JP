# ACSD-68636：別のストアから請求書を作成する際に、ギフト カードの E メール ヘッダーにストア所有者名が表示されない

ACSD-68636 パッチでは、別の店舗から請求書を作成した後、ギフトカードの電子メールテンプレートが電子メールヘッダーの管理設定からの所有者名を反映しなかった問題が修正されています。 この修正が適用された後、適切なストア所有者のメール情報がメールヘッダーに含まれるようになりました。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.74 がインストールされている場合に使用できます。 パッチ ID は ACSD-68636 です。 この問題はAdobe Commerce 2.4.9 で修正される予定であることに注意してください。

## 影響を受ける製品とバージョン

**Adobe Commerce バージョン用のパッチが作成されます。**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.8-p2

**Adobe Commerce バージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>このパッチは、新しい [!DNL Quality Patches Tool] リリースを含む他のバージョンにも適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html) で互換性を確認します。 パッチ ID を検索キーワードとして使用して、パッチを見つけます。

## 問題

別のストアから請求書を作成した後にギフトカードのメールが送信された場合、メールのヘッダーに、管理設定の所有者の名前が反映されていませんでした。

<u> 再現手順 </u>:

1. セカンダリ web サイト、ストア、またはストア表示を作成します。
1. **[!UICONTROL Sender Name]** と **[!UICONTROL Sender Email]** を、セカンダリ web サイトまたはストア範囲の **[!UICONTROL Stores]**/**[!UICONTROL Configuration]**/**[!UICONTROL General]**/**[!UICONTROL Store Email Addresses]**/**[!UICONTROL General Contact]** で変更します。
1. ギフトカード製品（仮想）を作成して、2 番目の web サイトに割り当てます。
1. ギフトカードをカートに追加し、2 番目の Web サイトに注文します。
1. ギフト カードの E メールを送信する注文を請求します。
1. ギフトカードの電子メールの送信者を確認してください。

<u> 期待される結果 </u>:

メールには、店舗固有の一般連絡先の値を使用する必要があります。

<u> 実際の結果 </u>:

デフォルトの送信者メールアドレスが使用されます。
