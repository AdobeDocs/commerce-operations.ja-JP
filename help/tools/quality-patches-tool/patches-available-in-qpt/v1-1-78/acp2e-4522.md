# ACP2E-4522：複数の買い物かごの結合または見積もり保存要求を同時に実行すると、quote_coupons テーブルで断続的に重複キーエラーが発生する

ACP2E-4522 パッチは、複数の買い物かごの結合または見積もりの保存リクエストが同時に実行されるときに、quote_coupons テーブルで断続的な重複キーエラーが発生する問題を修正します。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78がインストールされている場合に利用できます。 パッチ IDはACP2E-4522です。 この問題は、Adobe Commerce 2.4.9で修正される予定です。

## 影響を受ける製品とバージョン

**パッチはAdobe Commerceのバージョン**&#x200B;用に作成されました

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7-p7

**Adobe Commerceのバージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7 - 2.4.8-p4

>[!NOTE]
>
>このパッチは、新しい[!DNL Quality Patches Tool] リリースを含む他のバージョンに適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]：パッチの検索ページ &#x200B;](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=ja)で互換性を確認します。 パッチ IDを検索キーワードとして使用して、パッチを検索します。

## イシュー

複数の買い物かごの結合要求または見積もり保存要求を同時に実行すると、断続的な重複キーエラーが`quote_coupons` テーブルで発生します。

<u>複製する手順</u>:

1. 管理者サイドバーの&#x200B;**[!UICONTROL Stores]** > **[!UICONTROL Settings]** > **[!UICONTROL Configuration]** > **[!UICONTROL Sales]** > **[!UICONTROL Multicoupon Settings]**&#x200B;で、注文あたりのクーポンの最大数= 10を有効にします。
1. **[!UICONTROL Marketing]** > **[!UICONTROL Promotions]** > **[!UICONTROL Cart Price Rules]** > **[!UICONTROL &#x200B; Add New Rule]**&#x200B;の下にカート価格ルールを作成します。
   1. **[!UICONTROL Rule Name]**: TESTCOUPON2026
   1. **[!UICONTROL Active]**: **[!UICONTROL Yes]**、**[!UICONTROL Websites]**: **[!UICONTROL Main Website]**、**[!UICONTROL Customer Groups]**；該当するすべてのグループを選択します。
   1. **[!UICONTROL Coupon]**: **[!UICONTROL Specific Coupon]**; **[!UICONTROL Coupon Code]**: TESTCOUPON2026。
   1. **[!UICONTROL Uses per Customer]**：空白のままにします。
   1. **[!UICONTROL Conditions]**：条件が追加されていません。
   1. **[!UICONTROL Actions]** > **[!UICONTROL Percent of product price discount]**; **[!UICONTROL Discount Amount]** = 10.
1. GraphQLを使用して、ゲストカートに少なくとも7の数量の商品を追加し、クーポンを適用します。
1. GraphQL mergeCarts呼び出しを6回同時に実行します。

   ```
   mutation {
     mergeCarts(
       source_cart_id: "<GUEST_CART_ID>"
       destination_cart_id: "<DESTINATION_CART_ID>"
     ) {
       id
       items { id product { sku } }
       applied_coupons { code }
     }
   }
   ```

<u>期待される結果</u>:

GraphQLの結合でエラーは返されません。


<u>実際の結果</u>:

mergeCarts GraphQLのミューテーションはエラーを返します。
ログは、複数の結合または引用符の保存リクエストが同時に実行される場合、GraphQL エラーが`quote_coupons` テーブル （QUOTE_COUPONS_QUOTE_ID_COUPON_CODE）の重複キー制約の違反によって発生したことを示しています。

```
tail -n 200 var/log/exception.log var/log/debug.log | grep -E "Duplicate entry|QUOTE_COUPONS_QUOTE_ID_COUPON_CODE"
```
