# ACP2E-4395：ギフトカード製品は、グローバルスコープで金額を指定せずにAPI経由で作成できます

ACP2E-4395 パッチは、グローバルスコープで金額を指定せずにAPI経由でギフトカード製品が作成される問題を修正します。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.81がインストールされている場合に利用できます。 パッチ IDはACP2E-4395です。 この問題は、Adobe Commerce 2.4.9で修正される予定です。

## 影響を受ける製品とバージョン

**パッチはAdobe Commerceのバージョン**&#x200B;用に作成されました

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7-p8

**Adobe Commerceのバージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.5 - 2.4.8-p5

>[!NOTE]
>
>このパッチは、新しい[!DNL Quality Patches Tool] リリースを含む他のバージョンに適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]：パッチの検索ページ &#x200B;](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)で互換性を確認します。 パッチ IDを検索キーワードとして使用して、パッチを検索します。

## イシュー

ギフトカード商品は、グローバルスコープに金額を含めずにAPI経由で作成できます。

<u>複製する手順</u>:

1. 次の手順で使用するギフトカード金額属性IDを取得します。

   ```
   GET /rest/V1/products/attributes/giftcard_amounts
   ```

1. APIを通じてギフトカード商品を作成します。Web サイトの範囲でのみ金額が表示され、*すべてのWeb サイト* （`website_id` = `0`）の金額は表示されません。

   ```
   POST /rest/V1/products
   {
       "product": {
           "sku": "test-giftcard-bug",
           "name": "Test Gift Card",
           "attribute_set_id": 4,
           "status": 1,
           "visibility": 4,
           "type_id": "giftcard",
           "price": 0,
           "extension_attributes": {
               "website_ids": [1],
               "stock_item": {
                   "qty": 1000,
                   "is_in_stock": true,
                   "manage_stock": false,
                   "use_config_manage_stock": false
               },
               "giftcard_amounts": [
                   { "website_id": 1, "value": 50, "attribute_id": <attribute_id> }
               ]
           },
           "custom_attributes": [
               { "attribute_code": "giftcard_type", "value": "0" },
               { "attribute_code": "allow_open_amount", "value": "0" },
               { "attribute_code": "is_redeemable", "value": "1" }
           ]
       }
   }
   ```

<u>期待される結果</u>:

リクエストはHTTP 400で拒否され、メッセージ「*すべてのWeb サイトの範囲の金額を指定する必要があります」または「開封金額を許可する必要があります*」が表示されます。

<u>実際の結果</u>:

リクエストはHTTP 200を返し、グローバル *すべてのWeb サイト*&#x200B;の範囲に金額がないにもかかわらず、ギフトカード製品が作成され、無効な状態のままになります。
