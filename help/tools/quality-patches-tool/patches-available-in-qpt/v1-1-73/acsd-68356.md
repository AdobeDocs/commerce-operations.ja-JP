# ACSD-68356: [!DNL GraphQL] カートのクエリで、バーチャル見積もりに対して誤った割引額が返されます

ACSD-68356 パッチは、[!DNL GraphQL] カートクエリが仮想の見積もりに対して誤った割引額を返す問題を修正します。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73がインストールされている場合に利用できます。 パッチ IDはACSD-68356です。 この問題は、Adobe Commerce 2.4.9で修正される予定です。

## 影響を受ける製品とバージョン

**パッチはAdobe Commerceのバージョン**&#x200B;用に作成されました

* Adobe Commerce（すべてのデプロイメント方法） 2.4.8-p2

**Adobe Commerceのバージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>このパッチは、新しい[!DNL Quality Patches Tool] リリースを含む他のバージョンに適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]：パッチの検索ページ ](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=ja)で互換性を確認します。 パッチ IDを検索キーワードとして使用して、パッチを検索します。

## イシュー

[!DNL GraphQL]個の買い物かごのクエリで、バーチャル見積もりに対して誤った割引額が返され、対象外のバーチャル商品に割引が適用されます。

<u>複製する手順</u>:

1. 2つの仮想製品を作成します。
   1. `virtual1` SKU
   1. `virtual2` SKU
1. 買い物かごの価格ルールを作成：
   1. クーポンなしで自動的に適用するように設定できます。
   1. アクション：
   1. 製品価格割引の割合 – 20%。
   1. SKUが`virtual2`の買い物かごアイテムにのみ適用されます。
1. ストアフロントを通じて、新鮮な空の見積もりに商品を追加します。
   1. 最初に`virtual1` SKUを追加します。
   1. `virtual2` SKUを秒単位で追加します。
1. 次の[!DNL GraphQL] クエリを使用してカートの詳細を取得します。

   ```graphql
   {
     cart(cart_id: "cart_id") {
       email
       itemsV2 {
         total_count
         items {
           uid
           prices {
             total_item_discount {
               value
             }
             price {
               value
             }
             discounts {
               label
               amount {
                 value
               }
             }
           }
           product {
             name
             sku
           }
           quantity
         },
         page_info {
             page_size
             current_page
             total_pages
         }
       }
       applied_coupons {
         code
       }
       prices {
         discounts {
           amount {
             value
           }
           applied_to
           label
         }
         grand_total {
           value
         }
       }
     }
   }
   ```

1. **[!UICONTROL cart]** > **[!UICONTROL items]** > **[!UICONTROL prices]** > **[!UICONTROL discounts]**&#x200B;を確認してください。

<u>期待される結果</u>:

* `virtual1` SKUに[!DNL GraphQL]に表示されている割引はありません
* `virtual2` SKUには、[!DNL GraphQL]に表示されている割引があります

<u>実際の結果</u>:

* `virtual1`と`virtual2`は、[!DNL GraphQL]の応答で同じ割引額を誤って表示します。
