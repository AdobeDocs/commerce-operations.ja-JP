# ACSD-67424: `GET /carts/search`のREST API応答の`updated_at`値が[!UICONTROL Negotiable Quotes]の[!UICONTROL Admin Panel]の値と一致しません

ACSD-67424 パッチは、**[!UICONTROL Negotiable Quotes]**&#x200B;を使用する際に`GET /carts/search` REST API応答の`updated_at`値が[!UICONTROL Admin Panel]の値と一致しない問題を修正します。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.72がインストールされている場合に利用できます。 パッチ IDはACSD-67424です。 この問題は、Adobe Commerce 2.4.9で修正される予定です。

## 影響を受ける製品とバージョン

**パッチはAdobe Commerceのバージョン**&#x200B;用に作成されました

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7-p4

**Adobe Commerceのバージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7-p3 - 2.4.8-p2

>[!NOTE]
>
>このパッチは、新しい[!DNL Quality Patches Tool] リリースを含む他のバージョンに適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]：パッチの検索ページ ](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)で互換性を確認します。 パッチ IDを検索キーワードとして使用して、パッチを検索します。

## イシュー

`GET /carts/search` REST API応答の`updated_at`値が、[!UICONTROL Negotiable Quotes]を使用する場合に[!UICONTROL Admin Panel]に表示される値と一致しません。

<u>複製する手順</u>:

1. *会社*&#x200B;と&#x200B;*B2B見積もり*&#x200B;を有効にします。
1. [!UICONTROL Admin Panel]に新しい見積もりを作成します。
1. 見積に製品を追加し、**[!UICONTROL Send]**&#x200B;をクリックします。
1. 会社ユーザーとしてフロントエンドにログインし、**[!UICONTROL My Quotes]**&#x200B;に移動します。
1. **[!UICONTROL View action]**&#x200B;をクリックし、配送先住所を追加して、**[!UICONTROL Send for Review]**&#x200B;をクリックします。
1. [!UICONTROL Admin Panel]で、見積の配送方法を選択し、**[!UICONTROL Send]**&#x200B;をクリックします。
1. ストアフロントから&#x200B;**[!UICONTROL My Quotes]**&#x200B;に戻り、**[!UICONTROL Quote]**&#x200B;を選択して&#x200B;**[!UICONTROL Proceed to Checkout]**&#x200B;をクリックします。
1. 注文する。
1. データベース内の見積もりテーブルの`updated_at`値を使用して、REST APIを介して見積もり情報を取得します。

   ```text
   /rest/V1/carts/search?searchCriteria[filterGroups][0][filters][0][field]=updated_at&searchCriteria[filterGroups][0][filters][0][value]={QUOTE_UPDATED_AT}&searchCriteria[filterGroups][0][filters][0][condition_type]=gteq
   ```

1. API応答の`updated_at`値と[!UICONTROL Admin Panel]の[!UICONTROL Quotes] グリッドの&#x200B;**[!UICONTROL Updated at]**&#x200B;値を比較します。

<u>期待される結果</u>:

REST API応答の`updated_at`値と[!UICONTROL Admin Panel]値は同じです（タイムゾーンの違いを考慮）。

<u>実際の結果</u>:

REST API応答は、[!UICONTROL Admin Panel]の[!UICONTROL Quotes] グリッドの&#x200B;**[!UICONTROL Updated at]**&#x200B;値と一致しない誤った`updated_at`値を返します。
