# ACSD-69351：間違った web サイトに表示されるギフトカードの残高と有効期限

ACSD-69351 パッチは、割り当てられた web サイトの範囲を検証せずに、間違った web サイトにギフトカードの残高と有効期限が表示される問題を修正しました。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77 がインストールされている場合に使用できます。 パッチ ID は ACSD-69351 です。 この問題はAdobe Commerce 2.4.9 で修正される予定であることに注意してください。

## 影響を受ける製品とバージョン

**Adobe Commerce バージョン用のパッチが作成されます。**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7-p8

**Adobe Commerce バージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>このパッチは、新しい [!DNL Quality Patches Tool] リリースを含む他のバージョンにも適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=ja) で互換性を確認します。 パッチ ID を検索キーワードとして使用して、パッチを見つけます。

## 問題

割り当てられた Web サイトの範囲に応じてギフトカードの残高と有効期限が表示されない問題。

<u> 再現手順 </u>:

1. *Web サイト A* と *Web サイト B* を作成し、各 web サイトに対応するストアおよびストア表示を作成します。
1. **[!UICONTROL Catalog]**/**[!UICONTROL Products]**/**[!UICONTROL Add Product]**/**[!UICONTROL Simple Product]** に移動し、[!UICONTROL Name]、[!UICONTROL SKU]、[!UICONTROL Price] （例：*10.00*）を設定し、**[!UICONTROL Stock Status]** を *在庫* に設定します。 「**[!UICONTROL Product in Websites]**」セクションで、「*Website A*」と「*Website B* の両方を選択し、製品を **[!UICONTROL Save]** きます。
1. **[!UICONTROL Marketing]**/**[!UICONTROL Gift Card Accounts]**/**[!UICONTROL Add Gift Card Account]** に移動し、**[!UICONTROL Website]** を *Web サイト A* のみに設定し、残高を *100.00* に設定し、**[!UICONTROL Status]** を *有効* に設定し、将来の有効期限を選択して、ギフトカードアカウントを **[!UICONTROL Save]** し、生成されたギフトカードコードに注意してください。
1. *Web サイト B* のストアフロントを開き、前に作成した製品を買い物かごに追加します。
1. カート ページで [**[!UICONTROL Gift Card]**] セクションを探し、*Web サイト A* に割り当てられたギフト カード コードを入力し、ギフト カードを適用せずに **[!UICONTROL Check gift card status and balance]** を選択します。 残高と有効期限が表示されます。
1. または、*Web サイト B* で顧客アカウントにログインし、**[!UICONTROL My Account]**/**[!UICONTROL Gift Card]** に移動して、同じギフトカードコードを入力し、残高を確認します。 残高と有効期限が表示されます。
1. *Web サイト B* でチェックアウトに進み、同じギフトカードコードを適用しようとします。 Web サイトの範囲の検証が原因で、システムがギフトカードを正しく拒否します。

<u> 期待される結果 </u>:

残高は割り当てられたウェブサイトに限定され、システムはギフトカードを適用する際に次のエラーを表示する必要があります。*ギフトカードアカウントのウェブサイトを修正してください：X*。

<u> 実際の結果 </u>:

割り当てられた web サイトの制限に関係なく、ギフトカードの残高が表示されます。
