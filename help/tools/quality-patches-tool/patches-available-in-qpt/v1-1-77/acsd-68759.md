# ACSD-68759：ストアフロントに [!UICONTROL Date of Birth] が表示される場合、アラビア語（サウジアラビア）ロケールで顧客アカウントの作成が失敗する

ACSD-68759 パッチでは、**[!UICONTROL Create an Account]** が **[!UICONTROL Locale]** アラビア語（サウジアラビア）に設定され、*属性がストアフロントに表示される場合に* Customer **[!UICONTROL Date of Birth]** ページの読み込みに失敗する問題が修正されています。 このパッチは、[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77 がインストールされている場合に使用できます。 パッチ ID は ACSD-68759 です。 この問題はAdobe Commerce 2.4.9 で修正される予定であることに注意してください。

## 影響を受ける製品とバージョン

**Adobe Commerce バージョン用のパッチが作成されます。**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.7-p5

**Adobe Commerce バージョンとの互換性：**

* Adobe Commerce（すべてのデプロイメント方法） 2.4.4-p2 - 2.4.4-p16、2.4.5-p1 - 2.4.8-p3

>[!NOTE]
>
>このパッチは、新しい [!DNL Quality Patches Tool] リリースを含む他のバージョンにも適用される可能性があります。 パッチがAdobe Commerceのバージョンと互換性があるかどうかを確認するには、`magento/quality-patches` パッケージを最新バージョンに更新し、[[!DNL Quality Patches Tool]: Search for patches page](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=ja) で互換性を確認します。 パッチ ID を検索キーワードとして使用して、パッチを見つけます。

## 問題

**[!UICONTROL Create an Account]** がアラビア語（サウジアラビア）に設定され、**[!UICONTROL Locale]** 属性がストアフロントに表示される場合、**[!UICONTROL Date of Birth]** ページの読み込みに失敗し、`TypeError` が発生します。

<u> 再現手順 </u>:

1. 管理者にログインします。
2. **[!UICONTROL Stores]**/*[!UICONTROL Attributes]*/**[!UICONTROL Customer]** に移動します。
3. **[!UICONTROL dob]** 属性コードを編集します。
   * **[!UICONTROL Show on Storefront]** を *はい* に設定します。
4. 「**[!UICONTROL Save Attribute]**」をクリックします。
5. **[!UICONTROL Stores]**/*[!UICONTROL Settings]*/**[!UICONTROL Configuration]**/**[!UICONTROL General]**/**[!UICONTROL Locale Options]** に移動します。
   * **[!UICONTROL Locale]** を *アラビア語（サウジアラビア）* に設定します。
6. ストアフロントに移動します。
7. **[!UICONTROL Create an Account]** ページを開きます。

<u> 期待される結果 </u>:

**[!UICONTROL Create an Account]** が **[!UICONTROL Locale]** アラビア語（サウジアラビア）に設定され、*属性がストアフロントに表示されている場合&#x200B;***[!UICONTROL Date of Birth]** ページはエラーなしで読み込まれます。

<u> 実際の結果 </u>:

* **[!UICONTROL Create an Account]** ページが読み込まれません。
* 次のエラーがログに記録されます。

  ```
  [2025-10-31T02:14:35.029229+00:00] report.CRITICAL: TypeError: iterator_to_array(): Argument #1 ($iterator) must be of type Traversable|array, null given in app/code/Magento/Customer/Block/Widget/Dob.php:421
  ```
