# 概要：[!DNL Quality Patches Tool] （QPT） v1.1.75

このサブセクションでは、[!DNL Quality Patches Tool] （QPT） v1.1.75 で使用可能なパッチによって修正された問題について詳しく説明します。

QPT v1.1.75 には、次のパッチが含まれています。
1. **ACSD-68289**：単一のフィールドで条件を満たすことを要求するのではなく、すべての検索可能なフィールドをまたいで最小の一致条件がまとめて満たされた場合に、フルテキスト検索が一致する製品を返すようになった問題を修正しました。
1. **ACSD-68359**：多くの製品がカートに入っている場合に、長い URL が原因で **[!UICONTROL Pick in Store]** を使用したチェックアウト中にストアを選択できない問題を修正しました。 以前は、これにより、ストアセール中に生成された長すぎる URL が原因で 414 エラーが発生していました。
1. **ACSD-68451**：複数の web サイトで、会社の管理者が 1 つの web サイトにログインし、別の web サイトに無関係の会社を作成しても、その無関係の会社と誤ってリンクされる問題を修正しました。
1. **ACSD-68490**：設定可能な製品の作成時に、制限された管理者ユーザーに対して「**[!UICONTROL Add New Attribute]**」ボタンが表示される問題を修正しました。
1. **ACSD-68517**：カタログおよびカタログ検索ページのフォーム再送信エラーを修正します。
1. **ACSD-68573**：カテゴリ権限が顧客ウィッシュリスト項目に正しく適用されなかった問題を修正しました。 修正後、ウィッシュリストの項目は web とGraphQLの両方で正しく表示され、ページ分割されます。
1. **ACSD-68615**：処理された組み合わせに注文 ID がない場合に、在庫予約報酬 CLI で例外が表示される問題を修正しました。
1. **ACSD-68793**：有効な製品を共有カタログに割り当てる際に、それらの製品が誤って拒否される問題を修正しました。
1. **ACSD-68925**:GraphQL リクエストの応答が、HTTP 仕様におけるGraphQLと連携する問題を修正しました。 リクエストを解析できない場合、認証されていない場合、リクエストが解析された場合に一般的な問題が発生した場合は、4XX の応答コードが返されます。

左側のメニューを使用して、特定のパッチページに移動します。
