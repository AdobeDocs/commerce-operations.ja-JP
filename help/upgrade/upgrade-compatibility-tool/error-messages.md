# [!DNL Upgrade Compatibility Tool] エラーメッセージ

{{commerce-only}}

このエラーメッセージリファレンスは、 [!DNL Upgrade Compatibility Tool].

エラーメッセージは、レベル（重要な問題、エラーおよび警告）とタイプ ( コアコード、カスタムコード、GraphQLスキーマ ) 別に分類されます。 各タイプには、次の情報が含まれます。

- **エラーコード**:Adobe Commerceがエラーメッセージに割り当てた識別子。
- **エラーの説明**：エラーの原因の概要を説明する説明。
- **エラーが発生したアクション**：該当する場合、は、エラーのトラブルシューティングと解決に関するガイダンスを提供します。

## 重要な問題

### コアコード

これらのエラーは、一部のコアファイルが見つからないか、元のファイルと一致しない場合に報告されます。

| エラーコード | エラーの説明 | 推奨アクション |
| --- | --- | --- |
| 2001 | コアファイルが見つかりませんでした | を実行します。 `composer install` コマンドを使用して、プロジェクトのルートディレクトリからアクセスできます。 |
| 2002 | コアファイルが変更されました | を実行します。 `composer install` コマンドを使用して、プロジェクトのルートディレクトリからアクセスできます。 |
| 2003 | コンポーザーの依存関係がインストールされていません | コンポーザーの依存関係が見つからないと、問題が発生する可能性があります。 を実行して依存関係を復元 `composer require package_name`. |
| 2005 | コアフォルダが見つかりませんでした | を実行します。 `composer install` コマンドを使用して、プロジェクトのルートディレクトリからアクセスできます。 |

| エラーコード | エラーの説明 | 推奨アクション |

| 7010 | カスタム制約は、ターゲットバージョンで削除されたコア列を参照しています | 制約を削除するか、referenceColumn 属性を更新します。 |
