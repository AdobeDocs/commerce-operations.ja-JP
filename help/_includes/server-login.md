# サーバーログイン

1. サーバーにログインします。
1. 次に切り替え： [ファイルシステム所有者](https://devdocs.magento.com/guides/v2.4/install-gde/prereq/file-sys-perms-over.html).
1. アプリケーションを複製したディレクトリに移動します。 以下に例を挙げます。

   ```bash
   cd /var/www/magento2
   ```
