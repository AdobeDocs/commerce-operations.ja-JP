# サーバーログイン

1. サーバーにログインします。
1. 次に切り替え： [ファイルシステム所有者](../installation/prerequisites/file-system/overview.md).
1. アプリケーションを複製したディレクトリに移動します。 以下に例を挙げます。

   ```bash
   cd /var/www/magento2
   ```
