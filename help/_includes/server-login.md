# サーバーログイン

1. サーバーにログインします。
1. [ ファイルシステム所有者](../installation/prerequisites/file-system/overview.md)に切り替えます。
1. アプリケーションを複製したディレクトリに移動します。 以下に例を挙げます。

   ```shell
   cd /var/www/magento2
   ```
