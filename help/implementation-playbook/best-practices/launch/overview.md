# 立ち上げフェーズ

ローンチフェーズには、次のアクティビティが含まれます。

- 運用開始前と運用開始後の最終的なチェックリストレビュー
- 実稼動のデプロイメント
- セキュリティ設定
- サービスの検証
- パフォーマンスの監視
