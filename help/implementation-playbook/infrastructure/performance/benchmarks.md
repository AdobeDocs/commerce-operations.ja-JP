# ベンチマーク

カスタマイズと CDN キャッシュを追加しないクラウドインフラストラクチャ上のAdobe Commerceには、次のベンチマーク値が設定されます。

- 2 億件の有効な SKU
- 平均 API 応答時間（300 ミリ秒）
- 1 時間に 1,700 件の注文
- &lt;2s の平均ストアフロント応答時間
- 時間別ページビュー数 50 万件
